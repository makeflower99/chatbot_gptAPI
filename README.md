# 논리 기반 AI 코풀챗 
코풀챗은 ‘코드를 풀어주는 챗봇’, ‘코드의 풀이과정을 알려주는 챗봇’ 이라는 의미를 가진 서비스로, 중,고등학교 학생들에게 컴퓨팅 사고력과 문제해결 능력을 키워줄 수 있는 챗봇 서비스입니다.
